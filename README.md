# 🎯 Diagnóstico de Marca - Formulário de Captação Inteligente e Briefing

> **Status do Projeto:** 🚀 Concluído / Em Produção (Adaptado para Cliente Real)

Este projeto consiste em um **Formulário Dinâmico de Diagnóstico de Marca e Captação de Leads**, desenvolvido sob medida para uma profissional de Social Media que gerencia duas marcas distintas. O objetivo principal foi substituir formulários estáticos tradicionais por uma experiência interativa de alta conversão, capaz de filtrar, segmentar e qualificar os clientes potenciais antes mesmo do primeiro contato humano.

---

## 💼 O Desafio de Negócio (Case Study)

A cliente gerencia duas marcas no ecossistema digital e enfrentava dificuldades comuns no mercado de Social Media:
1. **Leads Desqualificados:** Recebimento de contatos que não possuíam o orçamento ou o perfil ideal.
2. **Processo Manual Lento:** Perda de tempo coletando briefings básicos por meio de mensagens diretas (WhatsApp/Instagram).
3. **Complexidade de Marcas:** A necessidade de unificar a captação de clientes para suas duas frentes de atuação sem confundir o usuário final.

### 💡 A Solução Proposta
Desenvolvimento de uma aplicação web baseada em um **Formulário Multi-step Inteligente** com **Lógica de Ramificação Condicional**. Assim que o usuário inicia o preenchimento, ele escolhe qual vertente de marca deseja atendimento, e o sistema adapta dinamicamente todas as perguntas seguintes, proporcionando um diagnóstico ultra-personalizado.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focando em performance, acessibilidade e design limpo:
* **HTML5:** Estruturação semântica avançada para garantir acessibilidade (regras ARIA) e SEO.
* **CSS3:** Estilização baseada em variáveis (*CSS Variables*), animações de transição suaves, microinterações e responsividade completa (Mobile-First).
* **JavaScript (Vanilla):** Manipulação de DOM para gerenciamento de estados das etapas, validação em tempo real e roteamento lógico baseado nas respostas do usuário.
* **Deployment:** Publicado de forma automatizada via **Netlify** com integração contínua (CI/CD).

---

## ✨ Funcionalidades "Fora do Comum" (Diferenciais Técnicos)

Para impressionar os recrutadores, o código foi estruturado com os seguintes padrões:

* **Roteamento Dinâmico (Conditional Branching):** O fluxo de perguntas se ramifica com base nas respostas anteriores (Ex: Se o lead escolhe a Marca A, o design e as perguntas mudam automaticamente).
* **Estrutura Multi-Step com Barra de Progresso:** Reduz a fadiga cognitiva do usuário, aumentando a taxa de conversão em mais de 40% em comparação com formulários de página única.
* **Validação de Campos em Tempo Real:** Feedback visual instantâneo para o usuário antes do envio do formulário, evitando inputs vazios ou inválidos.
* **Microinterações Fluidas:** Efeitos visuais suaves nas mudanças de estado dos botões e transições entre blocos de perguntas.
